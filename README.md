#  Academic Assistant & Exam Planner

This project is a Django-based web application designed for computer engineering students to track their study momentum and monitor upcoming exams with a real-time countdown.

## ‚ú® Key Features

* **Dynamic Performance Analytics:** Visualize completed study sessions with a weekly chart using Chart.js.
* **Exam Countdown Timer:** Track the time remaining for exams in months, weeks, days, and seconds.
* **Flexible Planning:** Create daily, weekly, and monthly task lists to organize academic goals.
* **Modern UI/UX:** Features a sleek dark mode with glassmorphism effects and advanced CSS animations for an optimal user experience.
* **Interactive Task Management:** A fast checkbox system to update task completion status on the fly.
